# Steps to run localy 

- Clone this repo

  ` git clone git@github.com:WadeNaughton/shopify_challenge.git`
 
- CD into the repository

  ` cd shopify_challenge.git`

- Open the repository

  -Ex: `atom shopify_challenge.git`
  
- Run bundle

  `bundle install `

- Run rails and go to localhost http://localhost:3000/

  `rails s `
